# 五筆字型

配方： ℞ **wubi98**

[Rime](http://rime.im) 五筆字型輸入方案

五筆字型 98 版

## 安裝

拼音反查、五筆拼音混合輸入依賴於

  - [袖珍简化字拼音](https://github.com/rime/rime-pinyin-simp) ℞ **`pinyin-simp`**

[東風破](https://github.com/rime/plum) 安裝口令： `bash rime-install https://github.com/hlizard/rime-wubi98 pinyin-simp`
